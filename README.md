# testtest


이 프로젝트는 Github에서 Repository를 사용할 때 관리해야 할 것들을 연습하기 위해 작성한Repository입니다

[See Demo] (https://github.com/Blooossom/testtest)

## Prerequisite

- Java(Jdk 11)
- SpringBoot(2.7.8 SNAPSHOT)
- Spring Data JPA
- MariaDB


## How to Start

``` shell
$ git clone {repo addr}
$ cd repo-setup-sample
$ npm install
$ npm start
```

- Blog
